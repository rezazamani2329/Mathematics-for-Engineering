# MATLAB-Engineering-Maths-Projects_1
MATLAB Projects from 'Mathematics for Engineers' course on coursera done in 2026 for MFE program at UC Berkeley. Topics: Bifurcation Diagram for the Logistic Map, Computation of the Feigenbaum Delta, Fractals from the Lorenz Equations, Bessel Function Zeros, Two-Body Problem, Two-Dimensional Diffusion Equation, Steady Flow and Unsteady Flow.
this course has five parts:
1- Matrix Algebra for Engineers
2- Differential Equations for Engineers
3-Vector Calculus for Engineers
4- Numerical Methods for Engineers
5- Mathematics for Engineers: The capstone Course 
here is the address for that: https://www.coursera.org/specializations/mathematics-engineers
